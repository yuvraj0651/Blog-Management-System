## 📌 Overview
A dynamic Blog Management System built to create, manage, and publish blog content efficiently. This application simulates a real-world content platform where users can perform CRUD operations on blog posts with a structured and scalable frontend architecture.

## 🛠️ Tech Stack
- React.js
- Redux Toolkit
- Tailwind CSS
- JavaScript (ES6+)
- REST APIs / JSON Server

---

## ✨ Features
- 📝 Create, edit, and delete blog posts (CRUD)
- 📄 Detailed blog view with structured layout
- 🔍 Search and filtering functionality
- 🗂️ Category-based blog organization
- 🔄 Dynamic UI updates using Redux Toolkit
- 📋 Form handling with validation
- ⚙️ Scalable component-based architecture
- 📱 Fully responsive design
- 🌙 Dark mode support

---

## 🧠 Key Learnings
- Managing dynamic content-based applications
- Implementing CRUD operations with Redux Toolkit
- Structuring reusable components for content systems
- Handling routing for detailed pages (slug-based)
- Designing clean and readable UI for content platforms
