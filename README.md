# 📝 Blog Management System

## 📌 Overview
A blog management system that allows users to create, edit, and manage blog posts with a structured and scalable approach.

## 🛠️ Tech Stack
- React.js
- Redux Toolkit
- Tailwind CSS

## ✨ Features
- ✍️ Create, edit, delete blogs
- 📄 Dynamic blog rendering
- ⚙️ Admin-style interface
- 📱 Responsive design

## 🧠 Key Learnings
- CRUD operations
- State management with Redux
- Dynamic content handling

## 🚀 Future Improvements
- User authentication
- Rich text editor
